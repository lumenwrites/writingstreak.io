## Introduction (About this Course)

Hello! In this video series I will show you how to create your first game using Godot Game Engine.

The game we will make is called Dodge The Ghosts, in this game the player will play as a treasure hunter, their goal is to run around the dungeon, collect the gold coins, and avoid being eaten by ghosts. The more gold the player can collect before dying, the higher their score will be.

The goal of this course is to help you to take the first steps on your jounery towards becoming a game developer.

You will learn all the key concepts you need to understand in order to make a game.

-   We will learn about Godot's UI.
-   We will create a character that can move around, controlled by the player.
-   We will create the enemies that can attack and kill our character.
-   We will learn a simple and convenient way to create levels using tile maps.
-   We will create simple pickups, the gold coins that increase your score.
-   We will learn how to save the high score to display player's record.
-   We will add simple animations, sounds, and visual effects.
-   We will create a simple UI.
    

In today's video, I will introduce you to Godot, we will download and install it, and we will import the assets you'll need to create the game.
